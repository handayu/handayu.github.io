OANDA Exchange Rates API client

OANDA 基于 Java 技术(FXTrade),可以跨平台交易,为零售客户和机构客户提供了一个尖端平台。该平台可以以一种具有竞争性的、实时的方式进入外汇,商品，货币，CFD市场。FXTrade 还在2001年推出时是首批全自动货币交易平台之一， 并提供标准的C++/C#接口用以自动化策略交易;

View Project

Bitfinex trade API

全球最大的数字货币交易所之一，提供全球绝大部分币币交易，合约交易，和交割合约的流动性，并未广大投资者，交易员提供 C++/C#/Python接口，以便于直接对接交易所进行自动化策略交易，尤其是多交易所之间的套利交易；

View Project

TraderTraining

交易员训练工具，旨在帮助交易员高效复盘，自动生成行情并完成实时交易，汇总交易报告，以便分析交易者交易策略的有效性；

View Project

MetCro-Stock

资金管理-凯利公式模拟-蒙特卡罗模型-风控控制；模拟多种不同的行情走势，在基于固定胜率以及固定盈亏比，仓位管理在凯利模式下的N次交易后的资金束， 用以管理和测算策略开仓的仓位以及在相应资金管理下的未来获得收益的概率值；
