  2. 源代码与使用

桌面Hook定点版本：

界面捕捉，定点写入文本内容，截图内容，图形识别完成APP与策略服务器交互，实现策略服务器运行状态，策略信号状态等自动化监控与运维；

View Project

桌面Hook可移动版本：

界面捕捉，非定点可移动微信位置写入文本内容，截图内容，图形识别完成APP与策略服务器交互，实现策略服务器运行状态，策略信号状态等自动化监控与运维；

View Project

WeChat - WebAPI版本：

非界面捕捉，直接使用微信WebAPI登陆，可实现任意后台监控运行策略服务器的状态，策略信号APP的推送与APP其他交互等(新申请的微信号可能会被禁止网页版本登陆， 官网对网页版本登陆做了限制，没有被限制的情况下，可基于此接口版本开发任意的前端界面用于策略服务监控)；

View Project

其他功能细节与技术栈

功能点:

  • 预警内容包括账户变更,实时成交信息,策略信号,服务器状态等等;
  • 客户端图形识别APP发送过来的信息,策略服务器进行消息回复与应答;
  • 网页爬虫获取华尔街见闻的365*24小时的资讯信息，并进行微信播报(由于华尔街见闻已下架,可在此框架下爬取其他公开资讯Web的需要关注的信息);
  • 抓取Multicharts-中国版主力合约维护的网页信息，进行换月微信提醒;
  • 定时发送服务器的相关状态(可定在开盘前);
  • 进程监控，监控给出的进程是否正常工作，并微信提醒;
  • 基于.Net平台C#的微信网页版API,交互过程可以使用fiddler工具自主分析RequestURL信息;
