  2. 数字货币程序化交易终端

主功能一：

提供完整的登陆,订阅,行情展示,K线展示,K线合成,指标加工,策略交易模块,并附带策略交易模板,可自行构建交易策略，完成数字货币交易,或 基于此框架进行二次开发,对接多套数字货币系统；

View Project

主功能二：

提供交易员下单面板,可进行合约交易快速下单;资金管理面板,包含资金,持仓,委托,成交的展示;

View Project

其他功能细节与技术栈

功能点:

  • 可使用Visual Studio或Visual Code,C#语言基于策略模板进行自定义策略开发
  • 提供几百种指标的计算模板,可自行实现自己感兴趣的指标用以展示;
  • 可开多张K线图,实现多策略,多周期,多图表交易,实现账户自动化管理;
  • 对接的Okex交易所接口为异步接口实现的并行操作,高性能处理前端展示与策略逻辑处理;
  • 事件通知框架,异构前端,业务，与其他底层接口对接逻辑,便于解耦与快速扩展功能;
  • 该框架可拿来剥离,自行解构对接其他交易所行情和交易业务,实现快速二次开发;
  • 提供行情本地模拟,交易驱动,用以测试策略交易和其他功能业务点;
  • 由于数字货币交易需要科学上网,需要自己配置科学上网模式才可以实盘使用,建议先本地模拟;
  • 功能插件式管理;
  • 提供技术对接咨询与交流;
